For this project to work properly we need to do once: 
1. Create a terminal and do:
sage -sh
pip install --user ncpol2sdpa
pip install --user picos
pip install --user cvxopt

2. Restart the worksheet server.

========
If there are problems (e.g. the worksheet server does not reset and worksheets don't work), an option is to remove all the contents from the ~/.local folder. 
=========

To view this project it is possible to:
1. upload the .ipynb files to a github project (or another webpage) 
2. go to http://nbviewer.jupyter.org/
3. type my user name, "mforets", and enjoy
